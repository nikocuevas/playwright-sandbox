# playwright-sandbox

## Installation
```bash
git clone https://github.com/nikocuevas/automation-test-sandbox.git
cd playwright-sandbox
npm install
```
***
## run e2e test
```bash
npm run test
```
***
***
## run demo tests
> 2 seconds delay was added on each test for demo purposes
```bash
npm run demo
```
***
***
## open report
```bash
npm run report
```
